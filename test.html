What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a webpage using elements represented by tags. Browsers interpret HTML to render content like headings, paragraphs, links, images, and more.

✨ Simple HTML example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>HTML is the ❤️ of the web.</p>
  </body>
</html>
🧱 Key building blocks:
  • Elements – defined by start & end tags: <h1> ... </h1>
  • Attributes – provide extra info: class="container"
  • Semantic tags – <article>, <section> improve accessibility
