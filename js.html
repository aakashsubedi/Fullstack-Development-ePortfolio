What is JavaScript?

JavaScript is the only programming language that runs natively in every web browser, enabling interactivity, dynamic content, and real-time communication across the web.

1 / VARIABLES

Variable Declarations

JavaScript provides three keywords for declaring variables — var, let, and const. Modern JavaScript (ES6+) strongly favors let and const over var due to block scoping and predictable behavior.

📦

const

Block-scoped. Cannot be reassigned. Use for values that shouldn't change.

🔄

let

Block-scoped. Can be reassigned. Use for values that will change.

⚠️

var

Function-scoped. Hoisted. Avoid in modern code — causes subtle bugs.

JavaScript 
// ✅ Modern ES6+ declarations
const courseName = "Full Stack Development";
let currentModule = 4;
let isCompleted = false;

// const with objects — the variable itself can't be reassigned,
// but the object's properties CAN be mutated.
const student = { name: "Alice", grade: 88 };
student.grade = 95;        // ✅ Allowed
// student = {};           // ❌ TypeError: Assignment to constant

2 / FUNCTIONS

Functions & Arrow Functions

Functions are reusable blocks of code. JavaScript has multiple syntaxes — traditional function declarations, function expressions, and the modern arrow function syntax introduced in ES6.

JavaScript 
// 1. Function Declaration (hoisted)
function greet(name) {
  return `Hello, ${name}!`;
}

// 2. Arrow Function (concise, no own `this`)
const add = (a, b) => a + b;

// 3. Arrow with block body
const formatScore = (score) => {
  if (score >= 90) return "A";
  if (score >= 80) return "B";
  return "C";
};

console.log(greet("Alice"));      // "Hello, Alice!"
console.log(add(5, 3));           // 8
console.log(formatScore(87));    // "B"
💡 Key Insight

Arrow functions do not have their own this context — they inherit it from the surrounding scope. This makes them ideal for callbacks but unsuitable as object methods.

3 / DOM

DOM Manipulation

The Document Object Model (DOM) is a tree-like representation of your HTML that JavaScript can read and modify. This is what makes pages dynamic and interactive.

Selecting Elements

JavaScript 
// Modern: querySelector returns the first match
const title = document.querySelector("h1");
const submitBtn = document.querySelector("#submit-btn");
const cards = document.querySelectorAll(".card"); // NodeList

// Modifying content and style
title.textContent = "Updated Title";
title.style.color = "#2E5E99";
submitBtn.classList.add("active");

// Creating and inserting new elements
const newItem = document.createElement("li");
newItem.textContent = "New list item";
document.querySelector("ul").appendChild(newItem);
4 / EVENTS

Event Listeners

Events are actions that happen in the browser — clicks, key presses, form submissions, mouse movements. JavaScript responds to these using addEventListener.

JavaScript 
const btn = document.querySelector("#myButton");
let clickCount = 0;

// Attach an event listener
btn.addEventListener("click", (event) => {
  clickCount++;
  btn.textContent = `Clicked ${clickCount} time(s)`;
  event.stopPropagation(); // Prevent bubbling
});

// Form submit event
document.querySelector("form")
  .addEventListener("submit", (e) => {
    e.preventDefault(); // Stop page reload
    console.log("Form submitted!");
  });
⚠️ Common Mistake

Always use addEventListener instead of inline HTML attributes like onclick="...". Inline handlers mix structure with behavior, making code harder to maintain.

5 / ARRAYS

Arrays & Higher-Order Methods

Arrays store ordered collections of data. JavaScript's built-in array methods — map, filter, and reduce — are among the most powerful tools in modern JS.

JavaScript 
const scores = [72, 88, 95, 61, 79, 92];

// map() — transform every element → new array
const doubled = scores.map(s => s * 2);
// [144, 176, 190, 122, 158, 184]

// filter() — keep elements that pass a test
const passing = scores.filter(s => s >= 75);
// [88, 95, 79, 92]

// reduce() — boil array down to a single value
const total = scores.reduce((sum, s) => sum + s, 0);
const average = total / scores.length;
console.log(`Average: ${average.toFixed(1)}%`); // "Average: 81.2%"
✅ Best Practice

Chain filter and map together for clean, readable data transformations: scores.filter(s => s >= 75).map(s => s + " pts")
